# 🚗 차량 번호판 OCR 분석기 (EasyOCR 기반)

이 프로젝트는 Python의 `cv2` (OpenCV), `easyocr`, 그리고 `PyQt5` 라이브러리를 사용하여 이미지 및 동영상에서 한국어 차량 번호판 텍스트를 인식하고 시각화하는 GUI 애플리케이션입니다.

**핵심**: 이 프로그램은 번호판 영역 탐지를 위해 **YOLO와 같은 별도의 객체 탐지 모델을 사용하지 않고**, EasyOCR의 내장 텍스트 탐지 기능에 의존합니다.

## ✨ 주요 특징

* **GUI 기반**: PyQt5를 사용하여 사용자 친화적인 데스크톱 애플리케이션 형태를 제공합니다.
* **미디어 지원**: 단일 이미지 파일 및 동영상 파일 (*.mp4, *.avi 등*)을 불러와 분석합니다.
* **고급 이미지 전처리**: OpenCV의 **CLAHE(대비 균등화)** 및 **모폴로지 연산**을 적용하여 번호판 텍스트의 윤곽을 최대한 강조하여 인식률을 높였습니다.
* **시각화**: 인식된 텍스트 영역에 박스를 그리고, PIL을 이용해 한글 텍스트를 이미지에 오버레이하여 표시합니다.
* **안정성 강화**: 라이브러리 로드 실패 시 프로그램이 비정상 종료되는 것을 방지하는 안전 코드가 적용되었습니다.

## ⚙️ OCR 파이프라인 (작동 방식)

이 프로그램은 단일 단계(One-Stage) OCR 방식을 사용하며, 영상 품질을 극대화하는 데 중점을 둡니다. 

### 1. 이미지 전처리 단계

번호판 인식률을 높이기 위해 다음의 순차적인 전처리가 적용됩니다.

| 단계 | 목적 | 기술 |
| :--- | :--- | :--- |
| **Grayscale** | 색상 정보 제거 | `cv2.cvtColor` |
| **CLAHE** | 저조도/그림자 환경 대비 개선 | `cv2.createCLAHE` |
| **이진화** | 텍스트와 배경 분리 | `cv2.THRESH_OTSU` |
| **Morphology** | 텍스트 윤곽 보강 | `cv2.MORPH_CLOSE` |

### 2. EasyOCR 실행 및 인식

전처리된 이미지는 EasyOCR의 내장 탐지 모델(`reader.readtext`)로 전달되어 텍스트 영역을 찾고 그 내용을 인식합니다.
